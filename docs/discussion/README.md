# Discussion

> Status: active directory
> Last Updated: 2026-05-18

Discussion docs hold unresolved or exploratory thinking before it becomes a decision, plan, capability packet, contract, or version scope.

Use this directory when:

- the product scenario is still unclear;
- the architecture trade-off needs thinking but no decision is ready;
- a user pain point may or may not enter scope;
- competitive/product positioning needs exploration.

When a discussion converges, route it to one or more of:

- [decisions/](../decisions/)
- [capabilities/](../capabilities/)
- [plans/features/](../plans/features/)
- [product versions](../product/versions/)
- [architecture versions](../architecture/versions/)

# Product Versions

> Status: active directory
> Last Updated: 2026-05-18

Product version packages define what a user-facing Plato version promises, what is in scope, what is out of scope, and how current system capability maps to that version.

| Version | Status | Architecture | Entry |
|---|---|---|---|
| Plato 1.0 | active | A1 | [1.0 Overview](1.0/overview.md) |

Each active or planned product version should include:

- `overview.md`
- `p0-scope.md`
- `gap-analysis.md`
- `acceptance.md`
